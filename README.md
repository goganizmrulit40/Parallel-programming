# Лабораторная работа №1
# Умножение двух квадратных матриц

## 1. Задание:
- Написать программу на языке C/C++ для перемножения двух квадратных матриц.
- Исходные данные: файл(ы) содержащие значения исходных матриц.
- Выходные данные: файл со значениями результирующей матрицы, время выполнения, объем задачи.
- Обязательна автоматизированная верификация результатов вычислений с помощью сторонних библиотек или стороннего ПО (например на Matlab/Python).

##  2. Реализация:
### 2.1 Алгоритм умножения
Для перемножения матриц используется классический алгоритм со сложностью O(n^3).
### 2.2 Генерация данных
Матрицы заполняются случайными целыми числами от 0 до 9.
### 2.3 Формат выходных данных
Все матрицы сохраняются в едином формате: числа через пробел, строки разделены переводом строки.
### 2.4 Измерение времени
Использована библиотека <chrono> для высокоточного измерения времени выполнения.
### 2.5 Объём задачи
Количество операций вычисляется по формуле: 2 * n^3 (n^3 умножений и n^3 сложений).
