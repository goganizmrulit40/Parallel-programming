# Лабораторная работа №2
# Параллельное умножение матриц (OpenMP)

## 1. Задание:
Модифицировать программу из л/р №1 для параллельной работы по технологии OpenMP. Провести серию экспериментов с:
- разным количеством потоков (1, 2, 4, 8 и т.д.), 
- разными размерами матриц (примерно 200, 400, 800, 1200, 1600, 2000),
- использовать фиксированное существующее количество вычислительных ядер - 6

##  2. Реализация:
### 2.1 Алгоритм умножения
Для перемножения матриц используется модифицированный алгоритм со сложностью O(n^3) - с порядком циклов i-k-j.
### 2.2 Измерение времени
Каждый эксперимент повторяется 3 раза для получения достоверных результатов.
### 2.3 Метрики
|Метрика |	Формула	|Описание|
|--------|-----------|-----------------|
|Ускорение|	S = T₁ / Tₙ|	Во сколько раз быстрее с n потоками|
|Эффективность|	E = S / n × 100%|Насколько эффективно используются потоки|
|Объём задачи	|2 × n³|	Количество операций|

## 3. Настройка OpenMP в Visual Studio
1) C/C++ → Language (Язык)
- OpenMP Support: Yes
- OpenMP Show Detailed Information: No
2) C/C++ → Command Line (Командная строка)
- Additional Options: /openmp
3) C/C++ → Optimization (Оптимизация)
- Optimization: Maximum Optimization (Favor Speed) (/O2)
- Favor Size or Speed: Favor fast code (/Ot)
4) C/C++ → Code Generation (Генерация кода)
- Runtime Library: Multi-threaded (/MT), т.к. у нас Release
- Enable Enhanced Instruction Set: /arch:AVX
5) Компоновщик (Linker)
- Linker → Command Line: /openmp
