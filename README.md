# Лабораторная работа №3
# Параллельное умножение матриц (MPI)

## 1. Задание:
Модифицировать программу из л/р №1 для параллельной работы по технологии MPI. Провести серию экспериментов с:
- разным количеством процессов (1, 2, 4, 6, 8, 12), 
- разными размерами матриц (примерно 200, 400, 800, 1200, 1600, 2000).

##  2. Реализация:
### 2.1 Алгоритм умножения
Для перемножения матриц используется модифицированный алгоритм со сложностью O(n^3) - с порядком циклов i-k-j - с распределением строк матрицы A между процессами.
### 2.2 Измерение времени
Каждый эксперимент повторяется 3 раза для получения достоверных результатов.
### 2.3 Метрики
|Метрика |	Формула	|Описание|
|--------|-----------|-----------------|
|Ускорение|	S = T₁ / Tₙ|	Во сколько раз быстрее с n потоками|
|Эффективность|	E = S / n × 100%|Насколько эффективно используются потоки|
|Объём задачи	|2 × n³|	Количество операций|
